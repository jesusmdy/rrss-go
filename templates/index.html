RSS Reader API

This is a simple RSS Reader API that allows you to get the latest news from the most popular news sites.

Getting started

  • users

    • PUT /users

      Creates a new user

    • GET /users (authenticated)

      Gets the authenticated user

  • feeds

    • GET /feeds

      Gets the feeds list

    • PUT /feeds (authenticated)

      Inserts a new feed to the user's feeds list. The body must contain the following fields:

  • feed follows

    • GET /feed-follows (authenticated)

      Gets the user's feed follows list

    • PUT /feed-follows (authenticated)

      Inserts a new feed follow to the user's feed follows list.

    • DELETE /feed-follows/{feedFollowId} (authenticated)

      Deletes a feed follow from the user's feed follows list.

  • posts

    • GET /posts (authenticated)

      Gets the user's followed feed posts list
